# json2rasanlu
### Nodejs tool to convert a json input file to RasaNLU template files
* domain.yml
* stories.md
* nlu_config.yml
* nlu_training.md
* nlu_model_training.py


# Install
```
git clone https://gitlab.com/dev4telco/json2rasanlu.git
npm install
``` 


# Run
```
./json2rasanlu.js sample.json
```

# Dependencies
- node v9.5
